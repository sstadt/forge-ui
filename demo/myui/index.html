Alert

The alert component is designed to have it's methods accessed via ref in a parent component. Calling the appropriate method attached to the ref will render an error in the box. Setting the error as a new type will clear the previous errors in the alert.

The following methods are available on the ref as shortcuts and each take a single string argument as the message to add to the alert:

  • message
  • primary
  • secondary
  • success
  • warning
  • alert

The following params can be passed to the component:

canClose
Boolean; defaults to false
Set to true to display the close button in the top right corner.

<alert v-ref:my-alert :can-close="true"></alert>
<v-input type="text" label="Alert Message" :value.sync="alertMessage"></v-input>
<button type="button" class="button" v-on:click="newMessage('message')">Default</button>
<button type="button" class="button" v-on:click="newMessage('primary')">Primary</button>
<button type="button" class="button secondary" v-on:click="newMessage('secondary')">Secondary</button>
<button type="button" class="button success" v-on:click="newMessage('success')">Success</button>
<button type="button" class="button warning" v-on:click="newMessage('warning')">Warning</button>
<button type="button" class="button alert" v-on:click="newMessage('alert')">Alert</button>


new Vue({
  el: '#alert_demo',
  data: {
    alertMessage: ''
  },
  methods: {
    newMessage(type) {
      this.$refs.myAlert[type](this.alertMessage);
    }
  }
});

<modal :show.sync="showModal">
  <div slot="header">
    <h3>Look, a Modal!</h3>
  </div>
  <div slot="content">
    <p>some super awesome modal content</p>
    <button type="button" class="button" v-on:click="showModal = false">Close</button>

</modal>

  }
