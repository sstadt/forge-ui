--- title: 'fRadio' id: 'vue-fradio' index: 9 ---

Renders a radio button input

Params

v-model
String; required
The selected value.
label
String; defaults to an empty string
Renders a label above the radio options is passed a string value.
name
String; required
The name for the radio button group.
options
Array; required
An array of options to render for this radio button group. Each array element should be an object with a name, label, and value.
required
Boolean; defaults to false
True if required.

selected: {{ selectedOption }}

 

<f-radio label="My Radio Group" v-model="selectedOption" :options="radioOptions" name="myRadioOptions"><f-radio>


new Vue({
  el: '#radio_demo',
  data: {
    selectedOption: '',
    radioOptions: [
      { name: 'radioOptionOne', label: 'One Option', value: '1' },
      { name: 'radioOptionTwo', label: 'Two Option', value: '2' }
    ]
  }
});
