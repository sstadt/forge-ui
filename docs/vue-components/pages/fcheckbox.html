Renders a checkbox input

Params

v-model
Boolean; required
True if checked; false if not
label
String; required
The label to render with the checkbox
required
Boolean; defaults to false
If true, shows an error when not checked; see forms

checked: {{ checked }}

 

<f-checkbox :checked.sync="checked" label="My checkbox"></f-checkbox>
