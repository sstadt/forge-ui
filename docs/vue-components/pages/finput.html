--- title: 'fInput' id: 'vue-finput' index: 6 ---

The fInput component provides a quick and easy way to add a standardized form input with automatic validation. The fInput directive can generate a textarea or any html5 input type. Validation occurs automatically on blur.

Params

v-model
String; required
The input value
type
String; required
The input type; set to textarea to render a textarea instead of an input
label
String
The input label text
name
String
The name attribute for the input
placeholder
String
The input placeholder text
required
Boolean; defaults to false
True if the input should be required; see forms
equalTo
String; defaults to null
Set to another observable to test equivalency

<f-input type="email" label="Email" :value.sync="email" />


new Vue({
  el: '#input_demo',
  data: {
    email: ''
  }
});
